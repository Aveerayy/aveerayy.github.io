# caeesar.org

This repository contains the GitHub Pages site for caeesar.org, a lineage-first catalog service that agents and LLM applications query.

## Product Positioning

caeesar.org is a governance and access layer between users, AI agents, and enterprise data systems.

- Connect data systems once
- Query through a governed semantic catalog
- Track lineage from source to response
- Enforce policies and auditability on every request
- Support both MCP-based and direct service integrations

## Website

The landing page in `index.html` explains:

- The lineage-first architecture
- Why provenance and trust should come before raw tool access
- How ETL and metadata pipelines feed the catalog
- A phased roadmap for production rollout

## Deployment

This is a static site intended for GitHub Pages deployment at [https://caeesar.org](https://caeesar.org).

## Local Development

1. Clone this repository
2. Open `index.html` in a browser

No build step is required.
